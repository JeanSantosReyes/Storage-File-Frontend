# File manager made with Bootstrap

Instalaciones:
- React Router Dom
- Bootstrap
- Sweet Alert
- Axios
- React Dropzone
- JS File Download
- React Table v8

```
npm install react-router-dom bootstrap sweetalert2 axios react-dropzone js-file-download @tanstack/react-table
```
